## 1. Цель работы

Данная лабораторная работа призвана напомнить основные команды ОС Debian/Ubuntu. Также она позволяет познакомиться с Docker и его основными командами.

---

## 2. Задание

Необходимо:

1. Запустить контейнер Ubuntu.
2. Установить Web-сервер Apache.
3. Вывести в браузере страницу с текстом **"Hello, World!"**.

---

## 3. Подготовка

Для выполнения работы требуется установленный Docker.

---

## 4. Выполнение работы

### 4.1 Создание репозитория

Создан репозиторий `containers04` и клонирован на локальный компьютер.

---

### 4.2 Запуск контейнера

Команда запуска контейнера:

```bash
docker run -ti -p 8000:80 --name containers04 ubuntu bash
```

**Разбор команды:**

* `docker run` — запуск нового контейнера
* `-ti` — интерактивный режим с терминалом
* `-p 8000:80` — проброс порта (localhost:8000 → контейнер:80)
* `--name containers04` — имя контейнера
* `ubuntu` — используемый образ
* `bash` — запуск командной оболочки

---

### 4.3 Установка Apache

#### Обновление пакетов

```bash
apt update
```

**Назначение:** обновляет список доступных пакетов.
**Результат:** загружается информация о пакетах из репозиториев.

---

#### Установка Apache

```bash
apt install apache2 -y
```

**Назначение:** установка веб-сервера Apache.
**Ключ `-y`:** автоматически подтверждает установку.

---

#### Запуск сервера

```bash
service apache2 start
```

**Назначение:** запускает Apache внутри контейнера.

---

### 4.4 Проверка в браузере

Открываем в браузере:

```
http://localhost:8000
```

**Результат:** отображается стандартная страница Apache (It works!).

---

### 4.5 Работа с файловой системой

#### Просмотр файлов сайта

```bash
ls -l /var/www/html/
```

**Назначение:** показывает содержимое папки сайта.
**Результат:** отображается файл `index.html`.

---

#### Изменение страницы

```bash
echo '<h1>Hello, World!</h1>' > /var/www/html/index.html
```

**Назначение:** перезаписывает файл `index.html`.

После обновления страницы в браузере:

**Результат:** отображается текст:

```
Hello, World!
```

---

### 4.6 Конфигурация Apache

```bash
cd /etc/apache2/sites-enabled/
cat 000-default.conf
```

**Назначение:** просмотр конфигурации сайта Apache.

**Результат:** отображается конфигурационный файл, содержащий:

* порт (обычно 80)
* путь к корневой директории (`/var/www/html`)
* параметры виртуального хоста

---

### 4.7 Завершение работы

```bash
exit
```

**Назначение:** выход из контейнера.

---

### 4.8 Просмотр контейнеров

```bash
docker ps -a
```

**Назначение:** показывает список всех контейнеров (включая остановленные).

---

### 4.9 Удаление контейнера

```bash
docker rm containers04
```

**Назначение:** удаляет контейнер.

---
